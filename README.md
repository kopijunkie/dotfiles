# kopijunkie/dotfiles

My personal dotfiles, managed by [chezmoi](https://github.com/twpayne/chezmoi)

## Usage

Checkout repo
```$ chezmoi init https://github.com/username/dotfiles.git```

Check diff(s)
```$ chezmoi diff```

Apply changes
```$ chezmoi apply```

Get latest changes + apply
```$ chezmoi update```
